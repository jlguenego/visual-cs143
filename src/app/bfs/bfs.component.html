Description

This animation shows the a very simple breadth first search in action.

We consider a tree of integer values. The root value is 3.

For each tree node, we can compute its children. In this example, the getChildren(node) function takes the node value integer N in input and returns 3 childrens with value respective N+1, N+2, N+3.
