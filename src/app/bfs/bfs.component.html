Description

This animation shows the a very simple breadth first search in action.

We consider a tree of integer values. The root value is 6.

We look in the tree for a node which value is > 10.

For each tree node, we must be able to compute its children. In this example, the getChildren function takes the node value integer N in input and returns 2 childrens with value respective N+1, N+2.

In this example, when the first node which value strictly greater than 10 is met, the search stops, and the stack is cleaned.
