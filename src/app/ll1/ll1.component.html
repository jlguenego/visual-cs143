Description

This animation shows a sentence parsing with LL(1) algorithm. You can see that there is no backtracking anymore. But the grammar needs to be LL(1) compatible.

We have a context free grammar (CFG) that is LL(1) defined as follows:

  • E => int
  • E => (E Op E)
  • Op => +
  • Op => *

We parse the following sentence:

(int + (int * int))
