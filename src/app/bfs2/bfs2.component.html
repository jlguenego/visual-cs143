Description

This animation shows a sentence parsing with breadth-first search.

Check sentential form prefix

Foreach sentential form, we check that it has the same prefix than the sentence. If not, then we do not check the children, because none of them can be the sentence.

Foreach sentential form, we also check that the sentential form length is not longer than the sentence. But this is done only if there is no rules with Îµ (empty rules).

Grammar

We have a context free grammar (CFG) defined as follows:

  • E => T
  • E => T + E
  • T => int
  • T => (E)

Sentence

We parse the following sentence:

int + int
