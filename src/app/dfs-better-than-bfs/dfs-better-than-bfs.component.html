CS143 Source

CS143 Top Down Parsing, part I. Slide 82.

Description

Example of nightmare grammar for BFS but partially solved for DFS:

  • A => Aa
  • A => Ab
  • A => c

We parse the following sentence (c followed by 6 a):

caaaaaa

The maximum stack size is considerably reduced, but the number of test is about the same.
