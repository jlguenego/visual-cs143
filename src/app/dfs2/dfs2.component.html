Description

This animation shows a sentence parsing with depth-first search, with some optimizations. This is the last example with a backtracking algorithm. Next examples will use predictive algorithm (LL1, etc.).

Improvements are:

  • 1 lookahead token with check if the lookahead is in FIRST(nt)
  • filtering production rules according the lookahead

We have a context free grammar (CFG) defined as follows:

  • E => T + E
  • E => T
  • T => int
  • T => (E)

Note that we have switched the first two rules to show the worst case.

We parse the following sentence:

int + ( int + int )
